# Beyond Surviving Landing Page

Upload all files in this folder to the root of your GitHub repository.

## Before publishing

Replace these placeholders inside `index.html`:

- `YOUR_CHECKOUT_LINK_HERE`
- `YOUR_FIT_CALL_LINK_HERE`
- `YOUR_INSTAGRAM_LINK_HERE`

Do not upload the ZIP itself to GitHub Pages. Unzip it first, then upload the contents.
